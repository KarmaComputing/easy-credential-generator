# Easy-crendential-generator

- Generates usernames
- Generates passwords


## Setup

```
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
cp ./app/.env.example ./app/.env
```

## Usage

```
import generate.py

### Generate username:

  generate.user(8,3,6)
  
    Word length up to 8
    Difficulty is 3 (Out of 5)
    6 words generated
```
